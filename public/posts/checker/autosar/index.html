AUTOSAR C++ 가이드라인#

AUTOSAR는 자동차 소프트웨어 개발을 위한 C++ 코딩 가이드라인을 제공합니다.

주요 버전#

  1. AUTOSAR C++ 14/17

    • C++14 표준 기반
    • MISRA C++를 기반으로 함
    • 자동차 소프트웨어 개발을 위한 안전한 C++ 코드 작성 규칙 제공

  2. AUTOSAR C++ 20

    • C++20 표준 기반
    • 최신 C++ 기능 활용
    • 향상된 안전성과 신뢰성 제공

가이드라인 특징#

  • 코딩 스타일: 일관된 코드 작성 규칙
  • 오류 방지: 잠재적 오류 예방 규칙
  • 보안: 보안 취약점 방지
  • 안전성: 자동차 소프트웨어의 안전성 강화

주요 목적#

  1. 안전성

    • 자동차 소프트웨어의 신뢰성 향상
    • 오류 가능성 최소화

  2. 유지보수성

    • 코드의 가독성 향상
    • 유지보수 용이성 증가

  3. 표준화

    • 일관된 코딩 스타일
    • 팀 간 협업 효율성 향상
